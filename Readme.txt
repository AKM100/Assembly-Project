Simple Assembly Calculator
 Simple Assembly Calculator project! This assembly language program is designed to provide a basic calculator functionality with support for addition, subtraction, multiplication, and division operations.

Features
Addition: Combine two numbers to find their sum.
Subtraction: Determine the difference between two numbers.
Multiplication: Multiply two numbers together.
Division: Divide one number bRun the program to see the result.

How to Run
To use the calculator, you need to assemble the source code using an assembler  and create an executable.
